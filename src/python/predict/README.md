# ПРОГНОЗИРОВАНИЕ
   - при поступлении новых данных не чаще 1 раза в месяц запускаем скрипт predict.R
   - после выполнения predict.R запускаем сприпт predict1.0.0.py
   - в базе данных (dima_db) появляеться таблица predict в которой находиться вся информация

# Документация к скриптам

- predict.R
    - Скрипт ничего не принимает он просто преобразовывает данне которые находяться в "6-ти таблицах"
    - Скрипт после обработки данных создает таблицу predictTable который потом используеться для прогнозирования


- predict1.0.0.py
    - Скприт ничего не принимает он скачавает обработаные данные с таблицы predictTable и переобучает модель по новым данным
    - После построения модели скрипт создает таблицу с нужными столбцами и записывает ее в базу в таблицу predict

- Замецание 
    - Модель возпращает данные только по тем товарам которые хотя бы 1 раз выставлялись 3 раза подряд
